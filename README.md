## 📌 Overview
This project is a **voice-enabled AI assistant** that can listen, understand, and respond in real time using **speech recognition + Generative AI (LLM)**.

It converts:
🎤 Speech → 🧠 AI Processing → 🔊 Voice Response

---

## ✨ Features
- 🎙️ Voice input using Speech Recognition
- 🤖 AI responses powered by Groq LLM
- 🔊 Text-to-Speech output
- 💬 Chat history with Streamlit UI
- ⚡ Real-time interaction pipeline

---

## 🛠️ Tech Stack
- **Frontend:** Streamlit  
- **Backend:** Python  
- **LLM:** Groq (LLaMA 3)  
- **Speech Recognition:** SpeechRecognition  
- **TTS:** pyttsx3  
- **Environment Management:** python-dotenv
